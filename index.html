Design Goals and Objectives

  • Create a user experience which leads people to enquire about the service
  • Build a responsive website that can viewed on all browsers and devices
  • Allow users to contact us through website
  • Allow potential employees to sign up via forms
  • Simple, Clean, Effective
  • SEO

Questions

  • What are we building?
  • Who are we building it for?
  • What features do we want to have?

Answers

  • A website to inform users about our services and to create a user experience which leads to people to enquire about the service
  • Potential customers who would be our target customers
  • Check User Stories

User Stories

  • As a user, I want to be able to find information about the service to see if it meets my customer needs
  • As a user, I want to to be able to find information about the company to see the legitimacy and history of the company ( Needs to be trustworthy based on the nature of our service)
  • As a user, I want to be able to find contact information for enquiry purposes
  • As a user(Potential Employee), I want to be able to find career oppourtinties so that I can apply for a relevant position
  • As a user, I want to be able to find location information
  • As a user I want to be able to request a brochure, to find more information about our services
  • As a user, I want to be able to request a call back (Feature) to speak to an employee
